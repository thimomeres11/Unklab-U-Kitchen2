# Folder Gambar Menu

Silakan tempatkan gambar-gambar menu di folder ini dengan nama file berikut:

## Makanan Berat:
- `nasi-omelet.jpg` - Nasi dengan omelet
- `mie-goreng.jpg` - Mie goreng komplit
- `bakso.jpg` - Bakso spesial
- `nasi-campur.jpg` - Nasi campur
- `bakso-2.jpg` - Bakso daging
- `mie-goreng-2.jpg` - Mie goreng spesial
- `nasi-omelet-2.jpg` - Nasi dengan omelet (alternatif)

## Minuman:
- `dessert-coklat.jpg` - Dessert coklat

**Catatan:** 
- Format gambar yang didukung: JPG, PNG, WebP
- Ukuran disarankan: 800x800px atau lebih besar (akan di-crop menjadi square)
- Nama file harus sesuai dengan yang terdaftar di menuData.ts
