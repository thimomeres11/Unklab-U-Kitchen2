# Instruksi Menambahkan Gambar Menu

## Lokasi Folder
Tempatkan semua gambar menu di folder: `public/images/menu/`

## Daftar File Gambar yang Diperlukan

### Makanan Berat:
1. **nasi-omelet.jpg** - Untuk menu "Nasi Goreng Spesial"
   - Gambar: Nasi dengan omelet di piring putih

2. **mie-goreng.jpg** - Untuk menu "Mie Goreng Komplit"
   - Gambar: Mie goreng dengan telur, rendang, sambal, dan acar

3. **bakso.jpg** - Untuk menu "Bakso Spesial"
   - Gambar: Bakso dengan isian sambal, daging, dan mie

4. **nasi-campur.jpg** - Untuk menu "Nasi Campur"
   - Gambar: Nasi campur dengan ayam goreng, urap, sate lilit

5. **bakso-2.jpg** - Untuk menu "Bakso Daging"
   - Gambar: Bakso dengan daging sapi dan tulang

6. **mie-goreng-2.jpg** - Untuk menu "Mie Goreng Spesial"
   - Gambar: Mie goreng dengan telur mata sapi dan rendang

7. **nasi-omelet-2.jpg** - Untuk menu "Nasi dengan Omelet"
   - Gambar: Nasi dengan omelet sayuran

### Minuman:
8. **dessert-coklat.jpg** - Untuk menu "Dessert Coklat"
   - Gambar: Dessert coklat dengan cornflakes dalam gelas plastik

## Cara Menambahkan Gambar

1. Salin file gambar Anda ke folder `public/images/menu/`
2. Pastikan nama file sesuai dengan daftar di atas
3. Format yang didukung: `.jpg`, `.jpeg`, `.png`, `.webp`
4. Ukuran disarankan: Minimal 800x800px (akan otomatis di-crop menjadi square)
